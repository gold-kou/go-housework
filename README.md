# サービス概要
ご家庭の家事を管理するソフトウェアです。

「誰が、いつ、何の家事」をやったかを管理することで、家事の負担を可視化することができます。

「自分ばかりが家事をしている」と悩んでいませんか？
まずは、「見える化」をしてみましょう！そこからパートナーへの相談に繋がります！
